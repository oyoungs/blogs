CGO 中对C语言的void *和void **的处理

1. void *与unsafe.Pointer

  • CGO中的 unsafe.Pointer与C语言中的 void *是对应的
  • Go中的结构体如果要传入 C语言的某个函数作为参数, 可以使用 unsafe.Pointer来转化

2. 对应的 void **只需要在现有的 void *类型(CGO中的unsafe.Pointer)的对象上取地址就可以了
